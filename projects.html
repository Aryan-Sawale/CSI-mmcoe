Brain Tumor Classification Using Transfer Learning
Made by Atharva Bhide

Brain Tumor MRI images have been classified into 4 classes by using Transfer Learning Models. The classes are-Glioma Tumor, Meningioma tumor, Pituatary tumor, No tumor Currently, the model is trained without data augmentation. In the future, data augmentation will be used on the training images to enhance the metrics. Resnet152V2 is the best performing model currently with 88.57% accuracy on training data and 88.16% accuracy on testing data

Neural Networks from Scratch
Made by Shrirang Mahajan

Basic neural network implementation in Python by using NumPy only. Includes: ReLU, Softmax Activation are added. Catogorical Cross Entropy Loss function is implemented. Optimizers - Adam, SGD, AdaGrad, RMSProp. Successfully classified breast cancer data with 94.29% accuracy.
